### OJ状态
1. AC (正确)
2. WA (结果错误)
	- 算法有问题
	- 输出形式不符(换行/空格)
3. TLE (运行超时)
	- 降低算法复杂度
4. RE (运行时错误)
	- 访问非法内存(**数组越界**)
	- 抛出异常(除0)
5. PE（输出格式错误）
	- *部分平台直接给WA*
6. CE（编译错误）
	- code存在语法问题
7. MLE(超出内存限制)
	- 别开超大数组

### 单个问题
#### 关于`n, m = int(input().split())`:
    **不可以**。`input().split()` 返回的是一个字符串列表（例如 `['1', '2']`），而 `int()` 函数不能直接接受列表作为参数。你必须使用 `map` 来将列表中的每个字符串分别转换为整数，或者使用列表推导式。

    正确的写法是：
    ```python
    # 使用 map
    n, m = map(int, input().split())
    
    # 或者使用列表推导式
    n, m = [int(x) for x in input().split()]
    ```

#### 使用 range() 函数进行数值遍历
range() 函数生成一个整数序列，常用于需要**循环固定次数**的情况。

示例:
```python
for i in range(5):
    print(f"当前i的值: {i}")
```
变种情况:
1. 指定起始、结束:
	```python
	for i in range(1, 6):
		print(i,end=" ")   
	#输出：1 2 3 4 5
	```
2. 指定步长:
	```python 
	for i in range(0, 10, 2):
		print(i,end=" ")
	# 0 2 4 6 8
	```
3. 反向遍历:
	```python
	for i in range(5, 0, -1):
		print(i, end=" ")
	# 1 2 3 4 5
	```
#### sum()函数：可迭代对象求和
1. 基本语法：`sum(iterable, start)`
	- `iterable`: 任何可迭代对象（如列表、元组、字符串等），其中的元素需要是数值类型。
	- `start`: 这个参数是可选的，用于指定累加的初始值，默认是 0。

	举例：
	```python
	sum([1, 2, 3])  # 输出 6
	sum([1, 2, 3], 10)  # 输出 16，因为 10 是起始值，1+2+3+10 = 16
	```
2. 题解:求一个整数各位数字之和:`dig_sum = sum([int(x) for x in str(i)])`
	>在这里，我们首先将整数 i 转换成字符串 str(i)，然后使用列表推导式将字符串中的每个字符转换为整数，并计算它们的总和。

#### 动态数组(array)
	如果需要存储大量的数值数据，并且希望节省内存，可以使用 array 模块。但对于大多数竞赛题目，使用列表已经足够。
	示例:
	```python
	import array
	a = array.array('i',[1, 2, 3, 4, 5])
	a.append(6)
	```
	>注意事项：
	array 需要指定元素的类型码，例如 'i' 表示整数，'f' 表示浮点数等。
	array 的操作方式与列表类似，但功能相对有限。
	{: .prompt-warning }

### 字符串
常用的 Python 字符串方法
1. 构造字符串
```python
s1 = "Hello"        # 直接赋值创建字符串
s2 = str(123)       # 从其他数据类型转换为字符串
s3 = "a" * 10       # 创建一个包含 10 个 'a' 字符的字符串
```
2. 字符串拼接（连接）
```python
s1 = "Hello"
s2 = "World"
s3 = s1 + " " + s2  # 使用 + 拼接
s1 += " Python"     # 使用 += 拼接
```
3. 获取字符串长度:`len(s)`

4. 字符访问
```python
s = "Hello"
c1 = s[1]    # 访问索引为 1 的字符，结果是 'e'
c2 = s[1:4]  # 获取子串，从索引 1 到 3（不包括 4），结果是 'ell'
```
5.**查找子字符串** 
```python
s = "Hello, World!"
pos = s.find("World")  # 返回子字符串首次出现的索引，如果找不到则返回 -1
```

6.**替换字符或子字符串**
```python
s = "Hello, World!"
s = s.replace("World", "Python")  # 将 "World" 替换为 "Python"(默认是全部替换，如果只想替换部分，可以加上第三个参数n，表示替换前n个)
```
7. 删除字符（通过切片）
```python
s = "Hello, World!"
s = s[:5] + s[7:]  # 删除从位置 5 到 6 的字符（删除 ", "）
```
8. 比较字符串 `==`

### ✨列表推导式
列表推导式是 Python 中用于生成列表的一种简洁语法，它允许你基于已有的可迭代对象快速生成新的列表。

基本语法：
```python
[expression for item in iterable]
```
**举例:**
```python
# 列表生成 1 到 5 的平方数
[x ** 2 for x in range(1, 6)]
```
