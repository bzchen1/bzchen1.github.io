## 1.codeFun Python 精品课

### 1.1 字符串切片 slice

基本语法:

```python
string[start:end:step]
```

- 提取区间 `[start,end)`,步长为 step。
- start(default 0),step(default 1)

**特殊情况:**

1. 取全部的串`s[:]`
2. 反转字符串`s[::-1]`

### 1.2 栈的实现

**栈(stack)的特性**: 后进先出(LIFO),只能在栈顶操作

**栈的操作**:

1. 入栈 push
2. 出栈 pop
3. 查看栈顶元素 top
4. 判断栈是否为空

**栈的应用:**

1. 函数调用
2. 表达式求值

**栈的实现：**

1. 数组

```python
s = []
s.append('a')    # push:o(1)
s.pop()          # pop:o(1)
s[-1]            # top:o(1)
is_empty = not s # empty(): o(1)
len(s)           # size()
```

### 1.3 pop()和 pop(i)

1. `pop()`：删除尾部元素,复杂度 o(1)

2. `pop(i)`：删除 index = i 的元素，并返回元素值，**由于需要移动元素，复杂度 o(n)**
   - 示例:
   ```python
   lst = [1, 2, 3, 4, 5]
   del1 = lst.pop(2) # 删除3
   del2 = lst.pop(-1) #删除5
   ```
